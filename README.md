# Campus Assistant - 校园助手系统

## 项目简介

Campus Assistant 是一个基于 Spring Boot + Vue3 + PostgreSQL 的校园助手系统，旨在为校园生活提供便捷的数字化服务。

## 技术栈

### 后端
- **框架**: Spring Boot 4.0.5
- **持久层**: Spring Data JPA
- **数据库**: PostgreSQL
- **工具**: Lombok

### 前端
- **框架**: Vue 3
- **构建工具**: Vite 5
- **UI 组件库**: Element Plus
- **状态管理**: Pinia
- **路由**: Vue Router 4
- **HTTP 客户端**: Axios

### 开发工具
- **版本控制**: Git
- **CI/CD**: GitHub Actions
- **包管理**: Maven (后端) / npm (前端)

## 快速开始

### 环境要求
- JDK 17+
- Node.js 16+
- PostgreSQL 12+
- Maven 3.6+
