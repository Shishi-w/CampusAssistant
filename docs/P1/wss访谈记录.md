# 访谈记录

### 访谈一
~~~ 
ss: 你在大学有用过或者了解过什么学生互助平台吗，可以发布跑腿接单、
学习求助、组队捞人等需求，等待其他同学帮助之类的
~~~  
   a：之前有在网上看到过类似有人去发起北京高校之类的(具体哪一方面的忘了)，但是没有使用，也没有关注了。
以上的功能我在我们学校的校圈上有用过，实名制的平台，只有本校学生可以使用，不过是匿名的

~~~
ss：你一般会发布什么需求，平台上有专门发布需求的板块吗，
有没有涉及支付报酬
~~~
  a：我平时发布的很少，其他人发布的很多，关于学习上的：本基 大创组队，咨询选课的问题，推荐的老师；二手旧物回收：出书，自用物品出二手，或者出掉多的票。还有单纯的摄影：可以分享校园风景图片，或者说约着一起拍照，找各种的搭子，跑腿。除了这些还有更深入的情感沟通：比如家庭问题的讨论，还有青年男女朋友恋爱的讨论。

  版块：不是特别固定，想发什么都可以。涉及报酬支付方面的话，平台是不需要，但是如果求助时候可以为别人提供小偿。
~~~
ss：你发布需求之后完成整个流程的过程顺利吗 有没有遇到什么问题 或者希望平台做出改善的地方
 ~~~
  a：还挺顺利的，但是有一点不好就是别人能看到其他人所有的评论以及发的帖子历史。希望这个能够设置时效性，这样别人就不会通过这样的方式去推测出这个人的身份。

~~~
ss：嗯嗯好的。为了鼓励大家使用平台我们会设置报酬机制，站在用户的角度你觉得是用户之间直接交易好，还是需要平台统一制定一个报酬管理机制好（或者说你觉得需要平台管理员对订单报酬进行审核管理吗）
~~~
  a：制定一个报酬管理机制可能会更加规范，对于我们用户来说这样可能更加科学。因为用户直接交易的话，定价也比较模糊，可能即使没有发生直接性的经济纠纷，但是心里面可没有底，不知道对方觉得这个价格是高了还是低了，而且大学生脸皮薄的话也不好意思讲价。如果有这种机制会更加规范，统一标准也比较便利。
~~~
ss：好的，订单完成后的评价环节设有用户打分功能，你对这个板块有什么想法吗，这个问题比较开放，随便说什么都行，或者可以提一些疑问
~~~
  a：这个其实设置很多人都懒得打分，每次完成一单就要打分还会觉得浪费时间。感觉也不用每一单完成之后就打分，也可以隔半年时间或者较为长时段收集这个打分意见反馈。不过这个是匿名的吗？匿名的会更好，如果有违反规则的最好有较为权威的证明然后才可以提供违规、侵害别人利益的人的信息，不然对用户的个人信息保管不好，很容易降低用户的信任度。
