# 访谈笔记

**问：** 在校园生活中，你最常需要他人协助的事务有哪些？可以先简单罗列一下。

**XC：** 上厕所没带纸时借纸，打球没带球时借球。

**问：** 详细说说一个令你记忆深刻的求助经历，顺利不顺利都可以。

**XC：** 可以是别人求助我的经历吗？

**问：** 可以。

**XC：** 主要是那个求助人的身份让我比较印象深刻，那我就说别人求助我的经历吧。就是有一天，我在学校旁边的全家吃汤面时，吃泡面时，还有一个长相像印度人的一个人，然后坐在我旁边。然后过一会儿，他就用有点儿生疏的中文问我说：“你可以说英文吗？”然后我就说OK。然后我们就开始用英文说。然后他说，他是印度的，来我们学校的，嗯，应该是历史还是什么专业的留学生，但是他我们学校给的那个帮助特别少，就是他们的那个校园卡什么的，好像理论上来说应该是跟我们一样的，只是支付方式什么的。但是问题是他好像没有他的微信，好像是没有实名认证，还是说他没有身份证，还是什么的反正在支付那有点问题，导致他虽然有校园卡，但是他没法在学校里吃饭，刷食堂的卡。然后他就问我说，知不知道在哪儿可以找到学校的那个电子的那个信息处？然后我就旁边找了张纸，然后画给他说去哪里哪里走，然后去跟他说，跟他说是什么问题。然后与此同时，还帮他搞定了他手机的NFC怎么设置。然后最后他就加了我的微信，然后走了。然后看他那个名字特别好玩，让我记忆特别深，他名字特别特别长，一大串英文字母看不懂，反正感觉留学生还挺有意思的。

**问：** 可以可以。你提到了你帮助别人的经历，是他向你求助的。你认为这次帮助别人，你最大的收获是什么？哪一点让你觉得最有价值？

**XC：** 我觉得最大的价值就是真的有去帮到人家，而且这个麻烦确实挺大的，因为他感觉中文也不太好。他要是在学校里直接去找别人问，然后别人看他不会说中文，可能一两句沟通不清。然后正好是我那天有空，然后我感觉就是那个让我感觉真正的有帮助到人家嘛。而且可能学校不太负责，所以也避免了这种一个惨剧的发生。哦，还有最大的收获就是知道了这个英语口语的重要性，因为跟陌生人沟通，可能不管什么国籍都可以用英语来沟通，OK，这也算收获吗？

**问：** OK，OK。所以说就是当别人真的遇到困难，可能这个求助于他人是这个仅有的选择的时候，你觉得你的一些小帮助能够真的缓解他们燃眉之急，很有成就感，或者说能满足自我需求这个需要。还有就是能够锻炼跟人交际交往的能力。

**XC：** 对，对，对你总结得特别好。就是一方面是能帮助到他人，一方面是自己也有这个成就感。而且还可以锻炼与人交往的能力，非常地上升高度。

**问：** 那除了这些精神层面或者说非物质层面的奖励之外，你觉得还有哪些方式比较能激励你去帮助别人？比如说一些能够兑换的积分或者是……

**XC：** 我觉得积分还挺好玩的，比如说搞一个认证系统。比如说你真的帮助到别人，那这个可以刷分啊——这个怎么办？怎么避免刷分这个行为？然后，如果别人给你帮助，他就给你认证一下，给你加1，太好玩了。还有啥能激励我？我感觉就是同伴的监督，或者是这个氛围吧，我靠，但这也是非物质层面的。物质层面的话，能不能给我搞个什么帮助到别人以后，然后别人跟那个辅导员汇报一下，然后给你整一个什么好人好事奖，然后给你加加分啥的。我不知道，也许有用。

**问：** 你说的对。从平台给出去的奖励应该是不能重复刷取的，就算平台要给奖励，也是那种全服共享一份奖励。

**XC：** 我觉得积分还挺好玩的，比如说搞一个认证系统。比如说你真的帮助到别人，那这个可以刷分啊——这个怎么办？怎么避免刷分这个行为？

**问：** 没事儿，你暂时不用考虑平台怎么实现，你就先想这个问题。如果说有积分或者是现金奖励，你帮助别人的意愿会不会更大？

**XC：** 会。

**问：** 你再想想你有没有求助的经历，比如说学习上面想要组队，或者失物招领，你用的树洞或者论坛上面发任何帖子应该都算求助吧？

**XC：** 有，在树洞上问学长学姐转专业的经历，他们给我了相关复习资料和PPT和面试经验。

**问：** 哦～当时你等了多久得到回应的？

**XC：** 一天吧。

**问：** 等急了吗？

**XC：** 急了，最后变有偿了。每人给了100。

**问：** 就是转专业全套资料呗？

**XC：** 还有面试资料。

**问：** 那是等了多久急了，然后转有偿之后又等了多久？

**XC：** 等了几小时急了，有偿之后一天就有人了。所以其实也不一定无偿就没人，只是当时懒得登了。

**问：** OK，了解。整个过程你觉得最麻烦、最困扰或者担忧的一个点是什么？包括你收到这个资料之后。

**XC：** 可能怕麻烦到别人，然后包括这种重复提问的方式也挺累的。当时有加3-4个人，然后不好意思拒绝别人，虽然第一个人已经回答得比较完整了，但是还是给剩下几个人红包了，听他们讲。

**问：** 你这个树洞平台是能保障用户都是本校学生，是吗？

**XC：** 是的。

**问：** 是什么方式啊？是学生证拍照认证了吗？

**XC：** 是账号的登录方式只有学号和密码。

**问：** 是那个学校统一认证的密码是吗？

**XC：** 对对。

**问：** 那这平台跟学校的数据库关联的？

**XC：** 对的。

**问：** 官方平台啊？

**XC：** 所以可以在上面查成绩课表啥的，半官方吧。授权算是。

**问：** 可能有社团，我猜的。

**XC：** 对的，就是有个社团。

**问：** 有的时候互助双方可能会产生纠纷，那么平台应该如何对互助结果进行判定，判定成功还是失败？

**XC：** 失败吧，严格一点。

**问：** 那就是倾向求助方了？结果失败就不给帮助方奖励。

**XC：** 确实。假设求助方都比较有良心，一般来说是这样吧，毕竟人家帮你了。

**问：** 你对校园互助平台还有什么建议吗？

**XC：** 希望奖励诱人一点。

**问：** 你可以畅想一下。

**XC：** 麦金卡，小红包，小黄车月卡，AI会员。

**问：** 你能接受平台出现广告吗？希望/不希望出现哪些类型的广告？

**XC：** 否。

**问：** 若平台有小额悬赏功能，你能接受的赏金范围大概是多少？

**XC：** 0-20。

**问：** 你是否愿意为信息置顶、精准匹配等增值功能付费？若愿意，可接受的单次付费金额是多少？

**XC：** 可，10。

**问：** 你认为哪些互助功能必须免费？哪些值得收费？

**XC：** 与学校和广大同学相关的求助可以免费，比较复杂耗时的求助（帮人配环境之类的）可以收费。

**问：** 什么是与广大同学相关的求助？

**XC：** 比如校园网不好使了怎么办，这种大家都会有的问题。

## R2

**问：** 在校园生活中，你最常需要他人协助的事务有哪些？比较愿意主动为他人提供帮助的事务有哪些？可以先简单罗列一下。

**R2：** 想象了一下大一到大三的自己。感觉会比较需要跨届引导，其实我觉得大一新生全在鼓楼其实导致大家和学长的联系很微弱，如果不太社交经常会陷入信息缺乏的局面。如果能整理出一个结构清晰的可编辑可精选的公告牌就好了。跨专业要注意什么，搬宿舍怎么安排，各个专业的课程常见疑问，有什么学分最好早点修掉。以及我以前有想过能不能组织学业辅导什么的，有偿或者无偿，不过这个通常各个专业内也有。介于校园集市和南哪助手之间的感觉？有哪些有趣的学生活动推荐。或许能把隐形的小兴趣团体在这里可见化，吸引趣味相投的人。

**问：** 酷啊，就是信息交流方面比较像论坛那个方向，还有小规模（可能1对1）的学业辅导。你有没有看过那个语雀文档？好像是南哪组织学长去编写的一份给新生参考的入学指南。性质可能跟你说的稍微有一点不同，不太像是互助平台，就是专门组织学长去总结一些信息吧。如果你看过使用过的话，可以说说看你使用过程中的一些想法，这个语雀的优缺点。没有使用过的话，也可以直接下一个问题，就是回顾一下有没有哪次你求助别人的经历比较印象深刻，说说整个流程。

**R2：** 这个我没看过耶。嗯嗯，我想的是比较有社区感的。就是南哪助手很清晰很系统，但可能没有那么灵活和精准。校园集市很灵活，但信息和经验是一次性的，很浪费。我想的是提问求助-回答-精选机制-整理机制。比如说精选靠点赞和置顶，整理靠大语言模型。下一个问题的话……我没有印象比较深的求助经历呢，但有一个印象比较深的被求助经历。

**问：** 哦也可以啊。

**R2：** 我以前申请过一次去德国慕尼黑交换的项目，虽然最后没去成。但从校内名额-校内流程-德国APS（一个审核程序）材料准备-签证材料准备的流程都走得差不多了。当时就有一个学妹顺着官网公示名单找到我，提出有偿咨询。其实就是问一下这些程序性的东西，材料怎么准备呀，德国的学制呀，语言问题呀。我觉得其实是自己查找会比较费神的东西。她当时V我50和我面聊了一下。我不知道这种咨询能不能平台化和简化。印象深可能是感觉从公示名单顺着学号姓名在QQ找到我然后发起咨询这样的行动很主动，让我觉得学妹很厉害。

**问：** 哦？当时被找到、被有偿咨询这个过程你主要会顾虑或者在意什么点呢？

**R2：** 其实我没什么顾虑的。我觉得我的探索经验变现了，而且帮到了别人，我觉得很开心，助人为乐者。我可能个人不擅长处理碎片信息吧，我觉得光是找齐网站，本校的、外校的、国家的、语言考试官方的、外交处的网站，就已经挺麻烦。后续整理信息也需要不少时间成本。我觉得那个咨询确实很有效的。

**问：** 你的独特经历或者资源能被特定有需要的人发现、正好帮上，这种感觉很不错是吧？

**R2：** 是。

**问：** 让特定的需求方和特定的帮助方匹配，平台的实现方式可以有很多种。你想象中有什么比较好的方式吗？（比如说最简单就是发布-接单。或者在你的案例中，学妹可以在公示名单上看学号，然后群发给实名过的用户？我刚想出来的。）

**R2：** 在匹配的问题上，我可能会担忧，如果太难被找到，可能会导致互助失效，但太容易找到，可能会让我容易被打扰。感觉设置陌生人打招呼回复前说一句话的机制会比较好。我觉得用户实名姓名和学号能直接搜索找到打招呼，就可以了。我觉得用户可以自己编辑简介，设置可检索的关键词，这样可能不需要查公示名单这个步骤了。

**问：** 嗯确实，所以就是有多个帮助方想要匹配时，帮助方可以发一条消息给求助方，求助方再挑选出来匹配或者细聊。就是小红书之类平台的“打招呼”是不是回复前只能说一句话？

**R2：** 嗯嗯。

**问：** 不赖，愿意帮助别人的用户公布、展示自己的特殊资源，这样。

**R2：** 对。
