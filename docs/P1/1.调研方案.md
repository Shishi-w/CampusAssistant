# 访谈提纲

## 零、引入
非常感谢你愿意抽出宝贵时间参与本次访谈～ 本次访谈主要是想了解大家在校园里互助的真实经历、遇到的困扰，以及对校园互助平台的真实需求，所有访谈内容仅用于项目调研，会严格保护你的个人信息，全程不会涉及敏感问题。在访谈过程中，你想到什么都可以直接说，包括正面回答、表示对某问题没有想法、对问题的疑问和建议等。
## 一、用户基础信息与过往经历
1. 在校园生活中，你最常需要他人协助的事务有哪些？最想主动为他人提供帮助的事务有哪些？（放到激励板块开头问 ）
2. 有没有委托熟人/陌生人帮忙的经历，详细说说过程是怎样的，你认为会经常有这种需求吗
3. 详细说说一个令你记忆深刻的经历,  
整个过程中，让你觉得最麻烦、最困扰的环节是什么？（引导用户主动提及信任、效率等问题）
## 二、竞品对比
1. 目前你解决校园互助需求的主要渠道是什么？现有渠道最让你不满意的地方是什么？
2. （可选）过去是否使用过班级互助群、校园跑腿小程序等类似互助工具？若使用过，使用频率及核心原因是什么？
3. （可选）你所在的班级/宿舍有固定的互助方式吗？对这种方式的满意度如何？
## 三、平台功能需求
1. 为保障互助安全，你认为平台需要身份认证吗？若需要，更接受哪种认证方式？
2. 平台是否需要即时聊天功能？
3. （可选）你是否需要平台设置评价体系和信用体系？
4. （可选）是否需要匿名发布/接收信息的模式？
5. （可选）你在平台查找信息时，是否需要分类、搜索、筛选、置顶等功能？
6. （可选）若有跑腿、代办等互助订单，你是否需要订单跟踪、进度提醒功能？
## 四、使用习惯与行为偏好
1. （可选）你一天中最可能使用平台的时段是什么？每次使用时长大概多久？一周使用几次？
2. （可选）发布一条求助/帮助信息，你愿意花费的最长时间是多久？
3. （可选）发布求助信息后，你期望的响应时间是？期望的解决时间是？
4. （可选）你是否习惯接收平台的消息提醒（如需求响应、活动通知）？
## 五、信任与安全需求
1. 与陌生人进行校园互助时，你最担心的问题（诈骗、爽约、信息泄露等）有哪些？
2. 平台需要提供哪些保障措施，才能让你放心使用？
3. 为保障互助安全，你认为平台需要身份认证吗？若需要，更接受哪种认证方式？
4. （可选）你是否愿意提供校园身份信息进行认证，以提升互助安全性？
5. （可选）遇到互助纠纷（如未完成服务、不支付赏金），你希望平台如何介入处理？
6. （可选）你对平台的信息隐私保护有哪些具体要求？
## 六、激励
1. 回顾（主动）帮助别人的一次经历，目的是什么
2. 对于提供帮助的人，哪种激励方式（积分/现金赏金等）更吸引你？
## 七、商业化与长期期待（可选）
1. （可选）你能接受平台出现广告吗？希望/不希望出现哪些类型的广告？
2. （可选）若平台有小额悬赏功能，你能接受的赏金范围大概是多少？
3. （可选）你是否愿意为信息置顶、精准匹配等增值功能付费？若愿意，可接受的单次付费金额是多少？
4. （可选）你认为哪些互助功能必须免费？哪些值得收费？
5. （可选）除核心互助功能外，你还希望平台承载哪些额外功能（如校园资讯、实习内推等）？
6. （可选）你是否希望平台延伸到毕业校友互助（如校友内推、职场互助）？
