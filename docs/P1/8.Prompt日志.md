### Prompt 1：评审需求规格说明书

**原始 Prompt**：

> pdf
>
> 现在你有三个文件 @830-1998.pdf @调研原始数据.md @需求规格说明书.docx @软件工程与计算II-问卷.xlsx 其中 @需求规格说明书.docx 是你要阅读并批阅的需求文档，@调研原始数据.md 和 @软件工程与计算II-问卷.xlsx 是该需求文档的原始数据，@830-1998.pdf 是需求规格说明的推荐模板，现在请你评审需求文档，给出修改建议到 md 文件。需要你关注的地方有：需求是否被恰当表达；需求是否来源于原始数据，是否有捏造成分；需求来源是否标注。注意，少用 read 工具会导致你报错。
